微信，微博获取biz和uid

1.请先安装谷歌浏览器Chrome，安装完后查询浏览器版本
2.安装Chrome版本对应的webdriver，并跟脚本放在同一目录下  http://chromedriver.storage.googleapis.com/index.html     
3.脚本默认将你需要查询的微博，微信列表放在d盘读取（微博：转uid和biz.xlsx--注意：替换KOL那一列即可）（微信：微信test.txt），结果也输出在d盘，如果想调整可以自己改脚本。。
4.微博脚本需要一个微博帐号密码，请在微博脚本里打开后，将提示区域改为自己的微博帐号和密码
5.脚本执行过程中都会自动打开浏览器进行操作，不用管，但也不要自己主动去关（它会自己关）
6.执行方法，打开CMD.先输入：d：  跳到d盘根目录
然后执行   python weibo.py
（注意请将两个脚本放D盘）
7.执行过程中需要全程联网，请确保良好的网络环境，如果断网会报错的，可能要重新执行。
